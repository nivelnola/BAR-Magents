"Magnetic Field Line Diagrams (Mostly) Work"
Eric Leong and Alan Wolf


Requirements:
Python 2.7+ - http://www.python.org/

Python packages (Note that these python packages may all be included as part of the Enthought Python Distribution, which we did not use):
Numpy - http://numpy.scipy.org/
matplotlib - http://matplotlib.sourceforge.net/
VTK - http://www.vtk.org/
wxPython - http://wxpython.org/
Mayavi2 via the Enthought Tool Suite (ETS) - http://code.enthought.com/projects/mayavi/


Instructions:
Run "calculate.py" to calculate data points.
Run "generate.py" to generate images.
Stitch images into a video using a tool like ffmpeg (http://ffmpeg.org/)
